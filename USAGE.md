# AI Chat DeepSeek - 使用指南

本文档提供详细的使用说明，帮助您快速上手 AI Chat DeepSeek 应用。

## 快速开始

### 第一步：获取 DeepSeek API Key

1. 访问 [DeepSeek 平台](https://platform.deepseek.com)
2. 使用邮箱或其他方式注册或登录账户
3. 进入 [API Keys 页面](https://platform.deepseek.com/api_keys)
4. 点击 "Create New Key" 创建新的 API Key
5. 复制生成的 API Key（以 `sk-` 开头）

### 第二步：启动应用

#### 本地运行

```bash
cd ai_chat_deepseek
npm install
npm run dev
```

然后在浏览器中打开 `http://localhost:5173`

#### 在线访问

如果应用已部署到在线服务，直接访问部署的 URL。

### 第三步：输入 API Key

1. 在 API Key 输入页面，粘贴您的 DeepSeek API Key
2. 点击 "开始聊天" 按钮
3. 应用将验证 API Key 并进入聊天界面

## 使用功能

### 选择 AI 智能体

在左侧边栏中选择不同的 AI 智能体：

- **跨境电商专家**：获取关于 TikTok Shop、选品、运营等方面的建议
- **健身教练**：获取健身计划、训练建议和营养指导
- **营销专家**：获取数字营销、品牌建设和用户增长策略
- **技术顾问**：获取编程、技术架构和开发建议
- **通用助手**：获取各种主题的帮助和建议

### 发送消息

1. 在底部输入框中输入您的问题
2. 按 **Enter** 键发送消息
3. 或点击 **发送** 按钮
4. 等待 AI 回复

**快捷键**：
- **Enter**：发送消息
- **Shift+Enter**：换行

### 查看对话历史

- 所有对话都会自动保存在浏览器本地
- 切换 AI 智能体后，之前的对话会被保存
- 再次选择同一个智能体时，会恢复之前的对话

### 清空对话

点击顶部的 "清空" 按钮可以清除当前 AI 智能体的所有对话记录。

### 退出登录

点击左侧边栏底部的 "退出登录" 按钮可以退出当前账户。退出后需要重新输入 API Key。

## 高级功能

### 上下文管理

应用会自动维护对话上下文，确保 AI 能够理解您的完整对话历史。

### 代码块显示

如果 AI 回复包含代码块（用 ``` 包围），应用会以代码块的形式显示，便于阅读和复制。

### 错误处理

如果发生错误，应用会显示错误信息，常见错误包括：

- **API Key 无效或已过期**：需要重新输入有效的 API Key
- **请求过于频繁**：请稍后再试
- **DeepSeek 服务暂时不可用**：请稍后再试

## 数据隐私

- 您的 API Key 只保存在浏览器本地存储中，不会上传到任何服务器
- 所有对话记录都保存在您的浏览器中
- 应用直接从您的浏览器向 DeepSeek API 发送请求
- 清除浏览器数据会删除所有本地保存的信息

## 最佳实践

### 1. 提供清晰的问题

给出具体的问题和背景信息，以获得更准确的回答：

```
❌ 不好：如何选品？
✅ 好：我想在 TikTok Shop 上销售美妆产品，目标客户是 18-30 岁的女性，预算 5000 元，如何选品？
```

### 2. 利用对话上下文

充分利用多轮对话的优势，基于 AI 的回答进行后续提问：

```
用户：如何开始做跨境电商？
AI：[回答]
用户：那选品有什么技巧吗？
AI：[基于前面的回答继续回答]
```

### 3. 尝试不同的 AI 智能体

不同的 AI 智能体有不同的专长，对同一个问题可能有不同的见解。

### 4. 定期保存重要信息

重要的对话内容可以手动复制保存，避免误删。

## 常见问题

### Q: 为什么我的 API Key 无效？

**A:** 可能的原因：
1. API Key 输入有误，请检查是否完全复制
2. API Key 已过期，需要在 DeepSeek 平台重新生成
3. API Key 没有足够的配额，需要充值

### Q: 对话记录会保存多久？

**A:** 对话记录保存在您的浏览器 localStorage 中。只要您不清除浏览器数据，记录就会一直保存。

### Q: 我可以在多个设备上使用相同的 API Key 吗？

**A:** 可以，但每个设备都需要单独输入 API Key。对话记录不会跨设备同步。

### Q: 如何导出对话记录？

**A:** 目前应用不提供直接的导出功能，但您可以：
1. 手动复制对话内容
2. 使用浏览器的开发者工具查看 localStorage
3. 截图保存重要对话

### Q: 为什么 AI 的回答有时不准确？

**A:** 可能的原因：
1. 问题表述不够清晰
2. 提供的背景信息不足
3. AI 模型的局限性
4. 某些特定领域的知识可能不是最新的

### Q: 应用支持哪些语言？

**A:** 应用界面支持中文。AI 的回答语言取决于您的问题语言。

## 技术支持

3. 确保您的网络连接正常
4. 参考 [README.md](./README.md) 中的常见问题部分
